# JavaScript Fundamentals

JavaScript is a dynamic programming language used for web development.
Variables are declared with let, const, or var. Use let for mutable values
and const for values that should not be reassigned.

Functions can be declared using the function keyword or arrow syntax.
Always use curly braces for block statements and semicolons to terminate
statements.

Arrays are zero-indexed and have a length property. Common array methods
include push, pop, forEach, map, filter, and reduce.

Objects use key-value pairs and can be created with literal syntax {}.
Access properties with dot notation or bracket notation.

Loops: for, while, do-while. The for loop has three parts: initialization,
condition, and increment.

Comparison: use === (strict equality) instead of == to avoid type coercion.
