# Assignment: JavaScript Basics

Create a JavaScript program that:
1. Defines a function calculateAverage(grades) that takes an array of grades
   and returns the average.
2. Defines a function getGradeLetter(score) that returns a letter grade
   based on the score: A (90+), B (80-89), C (70-79), D (60-69), F (<60).
3. Creates an array of student scores and calculates the class average.
4. Prints each student's grade letter to the console.

Requirements:
- Use let or const for all variable declarations (no var).
- Use strict equality (===) for comparisons.
- Handle edge cases: empty arrays, non-numeric values.
- Use meaningful variable names.
- Add comments to explain your logic.
